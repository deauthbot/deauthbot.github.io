Creating Verify Panel

Verification protects your server from raids and bot accounts.
When new member is joined, we'll quarantine them and wait for verify from him. Lets create a panel.
This command will create a new panel and automate verify actions. You don't need to do anything. /setup create

  • Verification made with text-based captcha.
  • Unverified users cannot access any channels (except: #verify) unless they verify itself.
  • We're also support custom title and description for your panel!

    • Editing Panel

    You can always edit the panel's title, description and colour easily. /setup edit


    Removing Panel

    Remove all panel components, disable verification. /setup remove

    • Captcha

    ・Captcha codes are generated randomly when user request a verify via panel. You can fully control this code generation-style & length etc.


    Captcha Mode

    ・Generation-style of captcha codes.

  • Basic (Default): a default 7 length random A-Z characters.
  • Hard Randomly A-Z characters, included symbols (>#@&/).
  • No Captcha: disables the captcha. The verify user in button click.

    • Captcha Length

    Custom length of captcha. By default it comes with 5 char length. You can set it up to 10 char length.

    Captcha Verify Fail

    Take a action when user failed to pass captcha.
    ・We offer 3 different modes. When user entered the wrong captcha code, the action you selected will be performed.

    Log Channel

    ・Get all action notifies about your whole server. Such as verification attempts etc.
    With this command you can set custom channel to receive Logs

    Quarantine Mode

    ・Which kind of users will be quarantined? filter the users that will be quarantined.

  • Only Users: only actual discord users will be quarantined.
  • Only Bots: whether only bots should be quarantined. Users won't affacted.
  • Both: bots & users will be quarantined.
